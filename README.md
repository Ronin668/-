# 甘特图
基于Excel表格所制作出的多功能甘特图
甘特图1是借助Qweb3.5-Plus大模型生成的甘特图，prompt为：根据该文档内容，以代码的形式生成甘特图，注意：任务按文档中的顺序排列。该图的不足之处在于：进度条是按照环节进行的分类，同一环节的不同任务颜色一致。优点在于：在图的右侧及上侧具有可交互按键，能够按照需求点击筛选内容。
甘特图2是借助Qweb3.5-Plus大模型生成的甘特图，prompt为：根据该文档内容，使用plotly 和 pandas 库以代码的形式生成甘特图，注意：任务按文档中的顺序排列；颜色根据十六进制颜色码自设，每项任务颜色设置禁止重复。经过一轮调试之后得到当前代码。该图该图很好的按照任务顺序排列，当鼠标悬停在进度条上时，会显示任务名称。鼠标悬停在某个进度条的左侧时，为该任务的开始时间，悬停在右侧时，为该任务的结束时间。在图的上侧具有可交互按键。
甘特图3是借助Gemini3Pro大模型生成的甘特图，prompt为：根据该文档内容，使用plotly 和 pandas 库以代码的形式生成甘特图，注意：任务按文档中的顺序排列；颜色根据十六进制颜色码自设，每项任务颜色设置禁止重复。经过一轮调试之后得到当前代码。该图很好的按照任务顺序排列，当鼠标悬停在进度条上时，会显示当前任务的名称、所属环节、起始时间、当前状态以及持续时间。该代码为我认为相对满意的一次尝试。
